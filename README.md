# Neovim setup

---

Neovim dotfiles with majority of code written in vimscript :D

## Dependencies
- curl
- bash
- git

## Setup

To setup this configuration, you will first install vim Plug *(you can use the script located in scripts/)*.

Update and install the plugins using Plug commands *(see *`:h Plug`*)*

* PlugInstall to install new plugins
* PlugClean to remove deleted plugins from the plug directory
* PlugUpdate to update installed plugins
